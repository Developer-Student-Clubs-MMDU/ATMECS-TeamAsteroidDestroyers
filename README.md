## Run Locally

Clone the project

```bash
  git clone https://github.com/Developer-Student-Clubs-MMDU/ATMECS-TeamAsteroidDestroyers
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```

Go back and Navigate to frontend Folder
```bash
  cd frontend
  cd arvis
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
